About This App

The Heart Stroke Prediction app is a simple machine learning web application that predicts the likelihood of a person experiencing a stroke. It uses key health indicators such as age, hypertension, heart disease, average glucose level, BMI, and smoking status to make predictions.

This app is built using Python and Flask for the web interface, and a trained Random Forest model for predictions. It is intended for educational purposes and to demonstrate how machine learning can be integrated into a web application.

Always consult a medical professional for real health concerns. This tool is not a substitute for professional medical advice.
